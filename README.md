# vue-xlsx-preview

## vue2集成luckysheet
```
1、实现纯前端xlsx文件预览
2、自定义xlsx内超链接跳转，解决luckysheetxlsx首次无法正确加载超链接问题
3、xlsx文件放在 public/ 文件夹下，以 doc.xlsx 命名
4、lickysheet相关的静态文件放在 public/lucksheetstatic/ 文件夹下，需要在 ，已在index.html中引入
```
